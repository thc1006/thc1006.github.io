Colosseum-ORAN-FRL-Demo

一個為 O-RAN 設計，端到端、可重現的離線聯邦強化學習研究框架。我們將真實世界的複雜數據轉化為可驅動 AI 創新的洞見。

v0.1.1 - 探索 GitHub

核心架構與價值

智能數據管線

將龐雜的 Colosseum 原始數據軌跡，透過自動化腳本轉換為乾淨、高效且附帶元數據的 Parquet 格式，為所有機器學習任務奠定堅實基礎。

高擬真數位分身

提供一個 Gym-style 的離線模擬環境。它作為真實網路的數位分身，讓您的 RL Agent 能在安全、可控且零成本的沙盒中進行訓練與評估。

分散式智能框架

內建的聯邦學習機制，完整模擬了 O-RAN 中多個 gNB xApp 的協同決策過程。這不僅是一個演算法，更是對未來 RAN 智能架構的具體實踐。

快速上手

只需三個指令，即可在您的本地環境中啟動並運行整個專案。

1. 安裝環境與相依套件

git clone https://github.com/thc1006/colosseum-oran-frl-demo.git
cd colosseum-oran-frl-demo
pip install .[dev]

2. 執行數據準備

python scripts/make_dataset.py --raw /path/to/your/raw_csvs --out src/colosseum_oran_frl_demo/data/processed

3. 啟動聯邦學習訓練

python scripts/train.py --rounds 10 --clients 1,2,6
