Colosseum-ORAN-FRL-Demo

一個專為 O-RAN 設計，端到端、可重現的離線聯邦強化學習研究框架

前往 GitHub 專案

專案簡介

在 5G/6G 和 O-RAN 的研究中，將 AI 演算法從理論轉化為可驗證的實驗是一大挑戰。本專案旨在填補此一鴻溝，提供一個基於真實世界 Colosseum 數據集的完整工具鏈，讓研究人員可以專注於演算法本身的創新，而非繁瑣的數據工程。

核心工作流程

本專案的核心架構可分為三大階段：首先，透過**數據管線**將原始數據進行處理與特徵化；接著，將處理後的數據送入**離線模擬器**中，作為聯邦式 AI 代理的訓練環境；最終，透過訓練得到能進行**動態網路切片管理**的智能模型。

主要功能

🚀 高效數據管線

內建腳本能將原始 Colosseum CSV 數據轉換為高效的 Parquet 格式，大幅提升數據載入與處理速度，並自動附加實驗元數據。

🔧 離線模擬環境

提供一個 Gym-style 的 SliceSimEnv 離線模擬器，作為真實網路的「數位分身」，讓演算法開發與原型驗證變得快速且經濟。

🤖 模組化聯邦學習

包含一個以 DQN 為基礎的強化學習代理，以及 FedAvg 聚合邏輯，完整模擬了 O-RAN 中多個 xApp 的分散式學習情境。

✅ 可重現性保證

透過標準化的套件管理 (pyproject.toml) 和自動化的 CI 測試流程，確保所有實驗結果都能夠被輕易地重現。

快速開始

只需幾個簡單的步驟，即可開始您的研究：

1. 安裝環境

git clone https://github.com/thc1006/colosseum-oran-frl-demo.git
cd colosseum-oran-frl-demo
pip install .[dev]

2. 準備數據

下載 Colosseum 數據集後，執行我們的數據準備腳本。

python scripts/make_dataset.py --raw /path/to/raw_csvs --out src/colosseum_oran_frl_demo/data/processed

3. 開始訓練

執行訓練腳本，開始您的第一個聯邦學習實驗。

python scripts/train.py --rounds 10 --clients 1,2,6
